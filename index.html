1) 사람 관리

전역 사람이 아니라 전체 사람 목록입니다. 한 번 추가하면 여러 시간표에서 검색해서 다시 선택할 수 있습니다.

2) 시간표 만들기 / 검색

시간표를 만들고, 아래 검색창으로 원하는 시간표를 빠르게 열 수 있습니다.

3) 참여자 검색 추가 / 전체 확정 일정

선택한 시간표의 참여자를 검색해서 추가합니다. 한 번 확정한 일정은 다른 시간표에도 자동 반영됩니다.

4) 불가시간 입력

참여자 중 한 사람을 검색해서 선택한 뒤, 드래그로 주간 고정 불가 또는 특정 날짜 불가를 입력합니다.

선택된 사람이 없습니다.
드래그 시작 칸이 비어 있으면 추가, 이미 막혀 있으면 해제됩니다. 같은 사람에 대해 율전/명륜 일정을 번갈아 입력할 수 있습니다.

5) 가능 인원 결과

선택한 시간표 참여자 기준으로 30분 단위 가능 인원을 계산합니다. 클릭하면 가능한 이름을 볼 수 있습니다.

1) 카테고리 관리

기본 카테고리를 수정하거나 새로 만들 수 있습니다. 삭제 시 연결된 물품은 미분류로 이동합니다.

2) 물품 등록 / 수정

처음 조사한 정보를 등록하고, 선택한 물품은 같은 자리에서 수정할 수 있습니다.

현재 모드: 새 물품 등록

3) 물품 검색

캠퍼스, 카테고리, 검색어로 목록을 빠르게 좁힙니다.

4) 물품 상세 보기

목록에서 물품을 누르면 상세 설명과 이미지를 확인할 수 있습니다.

물품을 선택하면 상세 정보가 여기에 표시됩니다.

1) 이동 조건 / 물품 검색

2) 이동 바구니
